🖥️ Windows Server 2016 və Active Directory (AD) Quraşdırılması

Bu səhifədə Windows Server 2016-nın quraşdırılması və Active Directory xidmətinin əlavə olunması prosesi animasiyalarla addım-addım göstərilir.

1️⃣ Sistem yüklənməsi və quraşdırma

Windows Server başladılır

Dil seçimi

Quraşdırma növü seçilir (Custom)

Desktop Experience ilə versiya seçimi

Quraşdırma davam edir

Yenidən başladılır

Administrator parolu təyin edilir

İlk giriş

Server istifadəyə hazırdır

2️⃣ Əsas Konfiqurasiya Addımları

Host adının dəyişdirilməsi

Firewall-un söndürülməsi

Uzaqdan bağlantı (RDP) aktiv edilir

IP ünvanına baxış

Statik IP təyin edilir

Saat qurşağının dəyişdirilməsi

Dəyişikliklər üçün restart

3️⃣ Active Directory Quraşdırılması

Rollar və xidmətlər əlavə edilir

AD DS rolu seçilir

Domain adı yazılır

Directory Services üçün parol təyin edilir

✅ Serveriniz artıq domen idarəetməsi üçün hazırdır!
